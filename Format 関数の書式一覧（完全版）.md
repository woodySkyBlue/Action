# 📘 **Format 関数の書式一覧（完全版）**

Format の構文は次の通りです：

```
Format("テンプレート {0:書式} {1:書式}", 値1, 値2)
```

Text 関数と同じ書式指定子を使えますが、**複数値を一つのテンプレートに埋め込める**点が強力です。

---

# 🧮 **1. 数値フォーマット（Number Format）**

## ■ 基本記号
| 記号 | 意味 |
|------|------|
| `0` | 必ず数字を表示（0 も表示） |
| `#` | 数字がある場合のみ表示 |
| `.` | 小数点 |
| `,` | 桁区切り |
| `%` | パーセント表示（値 ×100） |

---

## ■ よく使うパターン
| 書式 | 出力例（値 = 1234.5） |
|------|------------------------|
| `{0:0}` | `1235` |
| `{0:0.00}` | `1234.50` |
| `{0:#,###}` | `1,235` |
| `{0:#,###.00}` | `1,234.50` |
| `{0:0%}` | `123450%` |
| `{0:0.00%}` | `123450.00%` |

---

## ■ 小数点の制御
| 書式 | 説明 |
|------|------|
| `{0:0.0}` | 小数1桁固定 |
| `{0:0.000}` | 小数3桁固定 |
| `{0:#.###}` | 小数最大3桁（不要な0は表示しない） |

---

## ■ 正負の書式を分ける
```
Format("{0:#,###; -#,###}", Value)
```

---

# 🕒 **2. 日付フォーマット（Date Format）**

Format 関数は Text と同じ日付書式を使えます。

## ■ 基本記号
| 記号 | 意味 |
|------|------|
| `yyyy` | 西暦4桁 |
| `yy` | 西暦2桁 |
| `MM` | 月（2桁） |
| `M` | 月（1〜2桁） |
| `dd` | 日（2桁） |
| `d` | 日（1〜2桁） |
| `HH` | 時（24時間） |
| `hh` | 時（12時間） |
| `mm` | 分 |
| `ss` | 秒 |
| `fff` | ミリ秒 |
| `dddd` | 曜日（完全） |
| `ddd` | 曜日（短縮） |
| `mmmm` | 月名（完全） |
| `mmm` | 月名（短縮） |
| `AM/PM` | 午前/午後 |

---

## ■ よく使う日付パターン
| 書式 | 出力例（2026/01/06） |
|------|------------------------|
| `{0:yyyy/MM/dd}` | `2026/01/06` |
| `{0:yyyy-MM-dd}` | `2026-01-06` |
| `{0:MM/dd/yyyy}` | `01/06/2026` |
| `{0:dddd, mmmm d}` | `火曜日, 1月 6`（ロケール依存） |

---

## ■ 日付＋時刻
| 書式 | 出力例 |
|------|---------|
| `{0:yyyy/MM/dd HH:mm}` | `2026/01/06 05:28` |
| `{0:yyyy-MM-dd HH:mm:ss}` | `2026-01-06 05:28:00` |
| `{0:HH:mm:ss.fff}` | `05:28:00.123` |

---

# 💴 **3. 通貨フォーマット（Currency Format）**

Format 関数は Text と同じくロケール指定が可能です。

## ■ 日本円
```
Format("金額: {0:[$-ja-JP]#,### 円}", 123456)
```
→ `金額: 123,456 円`

## ■ 米ドル
```
Format("Price: {0:[$-en-US]$#,###.00}", 123456)
```
→ `Price: $123,456.00`

---

# 🌏 **4. ロケール（言語）指定**

Format の中でロケールを指定できます：

```
Format("Today is {0:[$-en-US]dddd, mmmm d}", Today())
```

---

# 🧭 **5. Format 関数の実務テンプレ（完全版）**

## ■ メール本文テンプレート
```
Format(
    "お疲れ様です {0} さん。\n本日の売上は {1:#,### 円} です。",
    User().FullName,
    Sum(Sales, Amount)
)
```

## ■ ギャラリーの行テンプレート
```
Format("No.{0} - {1} ({2:yyyy/MM/dd})", ThisItem.ID, ThisItem.Title, ThisItem.Created)
```

## ■ 多言語対応
```
Format(LocalizationTable.Greeting, User().FullName)
```
